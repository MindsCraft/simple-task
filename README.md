# Coup de Main

A modern, minimalist mobile application for tracking work time and managing teams.

## Features

- **Worker Mode**: Personalized QR codes for easy clock-ins and earnings tracking.
- **Employer Mode**: Integrated QR scanner for team management and project assignment.
- **Minimalist Design**: Clean, high-utility interface inspired by Apple Human Interface Guidelines.
- **Dark Mode**: Full support for system-wide light and dark themes.

## Tech Stack

- Ionic React
- Capacitor (Android/iOS)
- MLKit Barcode Scanning
- Vite
- TypeScript
